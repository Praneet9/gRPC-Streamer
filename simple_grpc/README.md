# gRPC-Streamer

gRPC server is written in Python and client is written in NodeJS

---

## Setup

Install python requirements
```(cmd)
$ pip install -r requirements.txt
```
```
$ npm install
```
---
## How to run

In a terminal run command
```
$ python3 server.py
```
In the second terminal run command
```
$ node client.js
```
